Con esta propiedad permite mostrar y ocultar la forma en que deseamos como aparece los menus, si queremos que se muestre en ventanas pequeñas del navegador o en lineas en ventenas mas grandes del tamaño de la ventana del navegador, esto ayudara a tener una pagina web mas dinamica.

En CSS es una de las mas complejas ya que establece el tipo de caja que genera cada elemento, la propiedas mas sencilla es el none ya que hace que no se visualice caja.

Para ventanas de navegador de menos de 800px, el CSS mostrará solo el formulario de selección dentro del elemento div con la clase de menú pequeño, para todas las ventanas de navegador más grandes, el CSS mostrará la lista ul dentro del elemento div con la clase de menú grande. Esto crea un efecto en el que la página se desplazará entre menús cuando la ventana del navegador cruce el ancho de 801px.

Para los tamaños de pantalla más grandes, puede usar la misma lista ul y usar la consulta de medios aún más para ofrecer un menú diferente simplemente cambiando el CSS y usando el mismo HTML.

Para el menú de tamaño mediano, use CSS para mostrar los elementos de la lista como una lista horizontal.

Esto convierte la lista en una lista horizontal. Queremos que esta versión de la navegación aparezca en las ventanas del navegador de tamaño medio. Colóquelo dentro de la consulta de medios entre 801px y 1024px.
